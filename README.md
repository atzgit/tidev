# Tidev  

Tidev app - Make your jobs easier !

# Function  
- Save time: Some settings on your computer are optimized with just a few simple steps.  
- System optimization: Some system features or other software are rarely used but make your computer slow. Tidev will solve this problem.  
- Utilities & personalization: The software is also integrated with a number of utilities to support your work. In addition, you can completely personalize the software's features if you like.  
### Feature details:
- Quick shortcuts to access legacy Windows settings (in the tray menu): Control Panel, Device Manager, Network and Sharing Center, Programs and Features, Sound, Windows Firewall (fast add or block applications), Group Policy, Registry, Services.  
- Application:  
  - T-Widget: displays computer parameters such as CPU, GPU, RAM, hard drive, temperature, network speed, ... time (supports multiple time zones in parallel with device time), user-defined quotes, weather forecast (requires the user's own Open Weather Map API key), air pollution (requires the user's own Air Visual API key).  
  - T-Finance: Electronic price chart widget from various sources such as Binance (supports API access to personal assets), OKX, Yahoo Finance, Dragon Capital, stock...  
  - T-Tools: View system information, set up a private proxy for applications, shake mouse to prevent sleep, display Caps Lock status, fix viruses that hide data on USB drives, automatically fix some computer errors, prevent automatic updates, prevent software from running automatically, volume up/down shortcuts, limit when applications can run, launch AutoIt script in real time and at startup (user-defined), set shutdown or sleep time, UnRAR lite, simple computer data backup...  
- App Force Disable: A strong measure to stop an application from running.  
- Cleaner: Cleans up junk files and folders.  
- Hosts Editor: Quickly edits the hosts file.  
- Third-party application integration: Process Explorer, Autoruns, Everything (search), FileTypesMan, Shell Menu View, WNetWatcher, etc.  
- Command features: Kcode, TiCrypt, WinSetOnTop, commands from Telegram bots, etc.
